# Quickstart: Publish on GitHub Pages

## Fastest path (no terminal)
1. Create a repo on GitHub (e.g., `grounded-phoenix-website`).
2. Download and unzip this package.
3. Upload **the contents of the `docs/` folder** to the repo (drag/drop in GitHub web UI).
4. Go to **Settings → Pages**:
   - Source: Deploy from a branch
   - Branch: `main`
   - Folder: `/docs`
5. Your site will publish at:
   `https://<your-username>.github.io/<repo-name>/`

## If you want a custom domain
Add it in **Settings → Pages → Custom domain**, then configure DNS at your registrar.
